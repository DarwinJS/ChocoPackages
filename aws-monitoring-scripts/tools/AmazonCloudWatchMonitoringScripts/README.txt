
README.txt for Amazon Cloudwatch Monitoring Scripts for Windows
===============================================================

The Amazon CloudWatch Monitoring Scripts for Windows are sample PowerShell scripts that demonstrate how to produce and consume Amazon CloudWatch custom metrics. The scripts comprise a fully functional example that reports memory, swap, and disk space utilization metrics for an Amazon Elastic Compute Cloud (Amazon EC2) Windows instance. You can download the Amazon CloudWatch Monitoring Scripts for Windows from the AWS sample code library.

For guidance on usage, please see http://docs.amazonwebservices.com/AmazonCloudWatch/latest/DeveloperGuide/
